Easy to install.2. Setting up Auto Facebook Popup is very easy.3. Advanced Customization Options are available on Backend.4. Can display popup in Homepage only. Two Options: Homepage or All Pages.5. Well Documented.== Installation ==This section describes how to install the plugin and get it working.1. Install the wordpress plugin firstYou can install it via Wordpress Installer or can download the file first then Upload it on 'wp-content' folder2. Activate the plugin3. You will see "FB Popup" settings from right side of your wordpress menu.4. Then you need to input your facebook information. So can display your Facebook Likebox.	i) Page Url: The URL of the Facebook Page	ii) Width: The pixel width of the plugin. Min. is 180 & Max. is 500	iii) Height: The pixel height of the plugin. Min. is 70	iv) Use Small Header: Use the small header instead	v) Show Timeline: Show or Hide Timeline.== Screenshots ==1. Screenshot of Frontend2.
